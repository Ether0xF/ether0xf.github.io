約瑟夫環。
以弗拉維奧·約瑟夫斯（Flavius Josephus）的名字命名的計算機科學和數學問題。關於這個多餘的背景介紹不說了，想稍微瞭解一下的可以查看這裏：Josephus Problem#History
在國內有些網站上也被叫作丟手帕問題。雖然不知道名字怎麼來的。

問題的描述很簡單。

  1. 一共有N個人圍成一圈。
  2. 從第一個人開始，數到K的時候將K位置上的人出列。
  3. 從K+1開始重新數。
  4. 重複2和3步驟，直到剩下一個人。
    求：最後一個人在原來隊列中位置，以及出列的順序。

分別用循環和遞歸兩種方法來做。語言是Ruby。
因為Ruby語言本身非常易讀直觀。完全可以當成英文來讀。而且最近Ruby學得高興，沒有寫Java的心情……
[toggle Title=”Josephus Problem#Iterative Version”]
用循環的方法來模擬這個過程非常簡單。
思路大概是這樣的。

新建一個列表a，長度為N。

計數器+1

如果 計數器 等於 K

++將K位置移出隊列。

++計數器=0

++出列計數器＋1

++輸出被移除的位置

指針移到K+1位置

如果 指針 指向了列表的末端

++指針回到0重新開始計數(使數組成為環形結構的重點)
循環直到隊列為空。輸出的最後一順位即為最後留下的人的原位置。
（+號是為了空格排版。b-quote內不支持縮進，一直不知道該怎麼弄。這個主題各個方面都好，唯獨b-quote讓我很煩惱。）

這個在Java中使用環形鏈表做起來很直觀。但是用Java和環形鏈表實現會費去太多別的精力在算法以外的東西上。不值得。
下面貼Ruby寫的循環版方法。

# Josephus problem solving by iterating analogy.
def josephusI(players, count_off)
  # set each element to be 1
  player_array = Array.new(players) { |element| element = 1 }
  count_out = 0
  counts = 0
  arr_index = 0

  until count_out > players - 1
    counts += player_array[arr_index] if player_array[arr_index] == 1
    if counts == count_off
      # set the dequeued element to be 0, i.e. it has been counted out
        player_array[arr_index] = 0 
        counts = 0
      count_out += 1
        puts "The #{count_out}th dequeueing: #{arr_index}"
      end
      arr_index += 1
    # if the index reaches the end, restart from 0.
    #   This makes the array to be circle.
    arr_index = 0 if arr_index == players 
  end
end

Kruth的某本書。類似的推導過程和sample code在Wikipedia上有。只不過這樣無法打印出列的順序。 推導的難點在於動態運算的 (f(n,k)+k) mod n。實際上這個是先深入到當f(1,k)，也就是隊列中只剩下一個人的時候，返回下標0。然後再遞歸向上計算在上一圈，上上一圈這個人的位置。 這個最好需要自己拿筆一層一層地跟蹤一遍。跟蹤的過程略長……今天太困了就不寫過程了。 以下是簡潔到寒磣的代碼。數學的力量真是偉大。 
# Josephus problem general case
# READMORE: http://en.wikipedia.org/wiki/Josephus_problem#The_general_case
def josephusR(players, count_off)
  if players == 1
     return 0
  else
    return (josephusR(players - 1, count_off) + count_off) % players
  end
end

# Josephus problem solving by iterating analogy.
def josephusI(players, count_off)
  # set each element to be 1
  player_array = Array.new(players) { |element| element = 1 }
  count_out = 0
  counts = 0
  arr_index = 0

  until count_out > players - 1
    counts += player_array[arr_index] if player_array[arr_index] == 1
    if counts == count_off
      # set the dequeued element to be 0, i.e. it has been counted out
        player_array[arr_index] = 0 
        counts = 0
      count_out += 1
        puts "The #{count_out}th dequeueing: #{arr_index}"
      end
      arr_index += 1
    # if the index reaches the end, restart from 0.
    #   This makes the array to be circle.
    arr_index = 0 if arr_index == players 
  end
end

# Josephus problem general case. Implementation in recursion.
# READMORE: http://en.wikipedia.org/wiki/Josephus_problem#The_general_case
def josephusR(players, count_off)
  if players == 1
     return 0
  else
    return (josephusR(players - 1, count_off) + count_off) % players
  end
end
