入职一个月

发表于 2016-11-17   |   分类于 The Journey   |  

没什么心得，有一些感想。

工作就是工作

我厂由于立地偏远但又幅员辽阔，提供有数量众多的员工宿舍并且已经形成了一个独特的小世界了。不管是我面试的时候还是办入职的时候还是入职后，几乎所有人都强烈推荐我住宿舍。
因为上班近，房租低。
但是我坚持在离市区更近的地方自己租了房子住。
公交车路上要20分钟左右，算上早上洗漱和早餐，加上我有起床困难和慢悠悠地喝咖啡的习惯，还有走去车站和等车的时间，相比住宿舍的人要早起很多才能上班不迟到。
反正听上去，并没有什么优势的选择。
但是对我自己来说，这个选择最大的好处就是，不必被拘禁在一个不可打破的小世界里。
规划生活作息的时间，用末班车的时间限制工作和加班的上限，效率地计划一天的工作，有机会接触本地市区的样貌和生活，了解起居周边，接触更多的陌生人，对本地真实的生活成本和物价能有更直观的感受。
而如果住在宿舍里，意味着不管任何时候，出入市区都不方便，每日在小小的世界里见到相同的人，去固定的地方，做差不多的事情，和住宿高中一样每日三点一线。即使在没有工作的日子，人也会半被迫地被停留在一个陆地孤岛中。
我觉得与从这里面得到的东西的价值相比，早起和等车的麻烦只是不足挂齿的代价。即使未来工作繁忙的时候，我可能会面临加班太晚没有公交车回家睡觉的窘境。
不管对事业有多高涨的热情，对未来有多么远大的志向，工作就是工作，人应该留出时间来生活。
职业的道路能走多远，要看在生活上能看得多远。反之，人生的视野，不应该因工作而狭隘。
一个想法。

有容乃大

成为社会人后，才体会到，世界是很大的。
即使曾经知道并理解这个道理，也在网上或者现实里接触过喜欢和不喜欢的形形色色的人。但是入职后依然会感慨，世界上是有各种各样的人的。
和以前相比，心生这种感慨的区别在于：曾经我能在网上，哪怕在现实里我也能仗着学生的身份，选择性地逃避我不喜欢或者觉得麻烦的社交，只参与我喜欢的小世界。
加上住宿时期时舍友有趣和善，大学时期一直独居，很少碰到非要在集体生活中将就的情况。就连大一住在恶评如潮的单元宿舍里时，我们单元的人也比隔壁要好很多至少不会半夜三点喝醉酒回来敲别人的窗户或者从来不收拾公共厨房。
走入社会环境后最大的感慨大概就是，世界很大，不由得你选择。
同期的同事来自全国各地，有不同的家庭背景和成长环境，不同的世界观和价值观，不同的学识教养，不同的爱好，不同的人际技能。虽然都还很和气，但是有些社交障碍的我还是能敏感地感觉到互相的世界的相异性，多少有一点不知所措。

不仅是办公室同事。入职体检的时候，去到地方，发现一个大屋子里坐了半屋子来应聘工厂普工的人。他们如何体检，我们便如何体检。几个人站成一排，都把袖子和裤腿挽起来检查是否有纹身和伤疤，内科检查也是三十秒就过，完全流水式。
我们未来都一样会是某条流水线上的一颗螺丝钉，然而即使如此其中依然有巨大的差异性，不仅是生活的世界，甚至是社会的阶层。比起他们，也许我还拥有更多的选择权。然而比起世界上的另一些人，我又感觉到我来到这里，是因为各种因缘差错使我没有能力有更多的选择权，始终无法摆脱一种「我可能无法成为更理想的人了」的恐惧感。

这大概就是被抛入社会的感觉。世界很大，由不得人选择。
然而并不是要愤慨，要自哀，拒绝接受现实给你的东西。不管是肉眼可见的差异性，以及不能选择的交际，还有喜欢和不喜欢的人和事，都要从接受开始。如果不想被大世界吞没，首先要从包容世界给你的东西开始吧。
海纳百川，有容乃大。

现学现卖

其实我经常做，还能装得和真的似的。
社会人老哥一直跟我说，其实大部分应届生什么都不懂。上班了后我发现，确实是这样。
更别提不少人的专业是通信或者自动化，并不是SE或者CS科班出身。只有一个同学做过单片机，调试板卡写驱动什么的还挺上手的。
其他人都处于，确实什么都干不了的状态，每日自学着C语言，无所事事看大佬们忙得飞天。
直到有一天我路过走廊正在聊工作的两个大佬身边时，大佬突然抓住我，听说你会Python是吧。
然后只玩过一个月Python的我就被赶鸭子上架了。
虽然只是写个内部用的工具和小脚本，但其中没接触过的东西也有，因为水平太渣思路堵住了又无处求助的时候也有，还有神特么在那种地方踩到坑的时候也有。但总算磕磕碰碰地把任务完成了。用朋友的话来说就是，打血源的新米猎人身上带着20瓶输血瓶竟然磕过了下水道。
写到第二个工具的时候，这个工具不算小，也比较严肃，而且内容完全陌生恐怕渣新人是hold不住的。大佬又本着不能让新人无所事事要锻炼一下的方针，加了一个同事进来，用大佬搭框架我们完成功能模块的方式来做。大佬说，同事不谙Python，做比较简单的那部分吧，你写难一点的这部分。
然后给我们两个一脸懵圈的听了两个下午大佬讲解思路，一脸茫然地回去了。
过了几天后，同事还在一个通用模块上纠结着。而我把他那部分的活都做了，开始着手查资料以应付大佬给我加塞的活了。
大佬有点苦恼，私底下和我说，「我觉得他可能理解不了我在说什么」。

不是想自夸，毕竟后来让我写PyQt的时候我踩了个不浅的坑，实在自夸不起来。
在各种论坛的帖子里，也常见新人抱怨自己并不是科班出身全靠自学大佬要求太高，大佬怎么什么都不教每天就让我print一下这个那个；或者是大佬一脸恨铁不成钢的样子说新人扶不上墙非科班出身不是借口，甚至还有大佬自己苦恼野路子出身和科班正牌到底差异在哪里。
作为一个渣新人，这种事情总是让我想很多。
工作和在学校不一样。很多东西都要能现学现卖，无穷尽的知识，有限的人生，是没有时间慢慢学的。陌生的内容，大佬飞快的思路，对新人来说就像明明已经到了体力的极限了还被马拽着往前跑，纠结而痛苦。
我就是纠结痛苦了那么久。也体会到了，他们要特意慢下脚步来等我，也是一件痛苦的事情。曾经我有一个本性温柔的大佬同学，本科的内容对他来说已经不是很难了。然而每次考前集体复习，他都要多次被打断，停下他的进度和思路，一边茫然地觉得「你们怎么这个都不懂」，一边努力组织好语言来给我们讲解，对他来说基础又简单仿佛都是本能和常识的一部分的一些知识点。
所以我想过很多，怎么让大佬和我自己都别那么痛苦。
然后我想差异在于对方法论的掌握。
不仅是学习的方法，也关乎把经验总结起来的方法。
技术经验是共通的，语言不过是门工具，思想才是精髓。从依葫芦画瓢的形式，到参悟模式和思想，这中间除了时间的积累，差的也许就是一个方法论。方法论掌握得越好，学得也越快。
然而方法论其实谁都可以尝试去掌握，只要学会思考就行了。只不过科班已经有了一套前人总结代代相传的经典方法和思想论，照本传授；野路子要自己摸到那套方法，可能需要一点时间罢了。
剩下的，需要的不是毅力，而是能战胜让人退缩的一切焦虑与恐惧的勇气。

Show Me The Code

就是在写第二个工具的时候，我怎么都跟不上大佬的思路了。无从下手，并且多次求助大佬，依然无法参悟。空坐了一天，一行都没写。
一度纠结又痛苦。
大佬对这个工具的想法也是处于一直在变化的状态，改动频繁，讲不清楚的地方很多，没想好的地方更多。加上时间紧迫，也没有什么设计什么需求什么文档，甚至没有可供调试的数据。
过了两天，大佬把我们叫过去，给我们看了一份代码的草稿，说，其实想你们写的大概就是这个样子的东西。
看到代码后一瞬间，好像觉得有点懂了。
拿了草稿回去后，开始读代码，改，写。写着写着，好像又不懂了。
然后又开始纠结，又写不下去了。
恰巧大佬经过我们办公室，过来问了一声，说先看看代码。然后指着屏幕上说，这里不是这样的，那个函数其实是这样用的，那个数据的结构其实是balabala所以要balabala……
指点一番，又拨开了一片云彩。
就这样我和大佬的两个脑洞不断碰撞，终于融合到了一起。
那感觉，真像宇宙诞生。

代码有时候也像文章，要从中挖出作者的心理和思想以及创作背景，是不太容易的一件事。一方面需要读者的学识三观审美相当，另一方面便是作者的隐晦的深层心理，甚至行文风格也会影响到阅读感受和理解。
但是代码总归是逻辑而客观的毕竟要不能正常运行就是狗屎最终在根本逻辑上，应该是要达到本质的统一和共识的。
所以无法互相理解的时候，甚至没有清晰的需求的时候，把脑洞写成代码，不要怂，正面撞。
说不定互相撞着撞着，就诞生了新的宇宙。
但是总有一股重构的冲动总是在蠢蠢欲动是为什么

近況

发表于 2016-10-05   |   分类于 The Journey   |  

とはいえ、特に報告しておきたいこともない。

一時期の苦痛は何時か引き退くのでしょう、と思ったら、また次の波が押し寄せてくる。こんなことや、あんなことや、それと焦りや鬱といった負の気持ちは、軽々しく口で叩いてもいいほど色褪せていない。
なんでもブログに書き込んだ若き私だったが、きっとこんなふうに抱え込んでいなかったのでしょう。
しかし、波が治まるまでに待つのみ、というのもどうかと思っている。こういう時こそ、何らかの無駄な足掻きで気持ちを少しでも動かせるのが大事かと、思う。何もやりたくない精がでないのが症状だから。
書くとしよう。

ちょっと健康問題が起こっている。
肺のあたりにチクチクする痛みがあるから気管支炎が悪化したかと検査を受けたら、炎症ではなく実は骨がいつのまにか折れてしまってしかももう二週間以上に放置されたままのせいだからという始末になった。
そもそも気管支炎の疑いは咳が中々止まないからのものだ。今度こそ病因を探り出し一発で治ろうと診察に行ったら、咳喘息だ静養せよそれと月一回再処方することは忘れないでねと告げられた。
体調は元々優れるものではないが、これはちょっと想定外だ。
幸い、どれも軽症で、そのうち治るでしょう。
だがこの一件で、もう若くないなーと再び実感してしまった。
優れないが、病弱なほどもない。体力は弱いけれど、ちょっとする不調があってもなんとか治るという半端ない回復力は、まさに活気あふれる若人の特権なのだ。
今や、夜更かしなんて身勝手なことはほどほどにせねば、すぐ体調に響いてしまう。回復力の特権が失った以上、くれぐれも注意しないと、と自分に言い聞かせ始めるのだ。

内定をもらったが、入職の手続きに面倒事があった。
この先に何がどうなるかは全く見えてこないから、鬱。
行き先は一応都会だが、会社は何故かとんでもない僻地にある。
アパートを先に押さえないとまずいことになりそうだ。
こういうことに気力が消耗される一方だから、最近プログラミングに気が向いていなかった。
代わりにペルソナ5にハマってしまった。こういうRPGが苦手だけれど、ストーリーとシステムの面白さのおかげで特に難がなかった。
やがて90時間弱でクリア。奥村嬢は実にかわいい。
友人のオススメでP4Gもやってみたけれど全く進まない。ダンジョンの探索を繰り返すゲームの旨さはイマイチわからない。

気が向いたら、服薬リマインダーアプリを自力で作ってみたい。吸入ステロイド薬の管理と記録のためでもあり、こういうアプリを一応捜してみたが、どれもイマイチだ。簡単なものでいい。いっそ自分で作ろうという発想についた。
スマホはAndroidに変えたから、Androidで作ると考える。
いい練習になるとは思う。

近況報告はこれで。
少し気を晴らしたので、書いた甲斐があった。
年明けに「今年は何があっても100Freshを50枚目までに頑張る」という願いを書き込んだ覚えはあるが、心の花が枯れたせいでとても描ける状態ではなくなった。
まずは一枚の落書きを完成させる、というのが今の目標というか願望というか。
とにかく、頑張らない。自然体でいこう。

R Programming - Subsetting

发表于 2016-07-30   |   分类于 Notes   |  

前段时间鬼使神差地在Coursera上学了一门R Programming。作为R入门来说通俗易懂，可搭配R Programming for Data Science食用。这本书本身也可以当作cookbook来用。初学者还不熟练的时候突然忘记某个函数怎么写时可以翻开看看压压惊。
要说课程的不足之处就是，感觉面向统计学家多于程序员。（毕竟开课的是统计专业）有些programming上想知道的细节都比较轻描淡写。
就连在写笔记的时候，信手就写，越写越自我怀疑，去查了文档后发现很多事情和我以为的不太一样，和教的，也不太一样。
所以查文档很重要。
很重要。
很重要。

这根本的原因，也许是因为R语言中对于表格的处理和计算都非常直观简洁好用，以至于太简洁太直观，初学容易飘飘然，用多了也会容易忘乎所以，以至于越用越不安。
加上语言本身也正处于高速变化和发展的阶段，不经意间还会碰到一些旧版本的历史问题。等到用得飘飘然，忘记要细究原理的时候，往往就是要栽坑的时候。

所以越是简洁直观节省麻烦，越要注意弄清楚它的是非和限制，不能忘记基础和根本，不要乐不思蜀。
事情光看表面，会容易出问题。

Subsetting其实很基础，但之所以写下这篇略显啰嗦的笔记，不是因为弄懂了，正是因为全是初学时栽过跟头的地方，才要记下来。
不知何时会拿出这篇笔记复习的未来的我，切记切记。

用Git管理FTP上传

发表于 2016-07-12   |   分类于 Notes   |  

事情是这样的

陆陆续续给这个Blog添加了一些新东西。比如About页和评论模块等等。

然而最主要的，还是解决了先前说的FTP同步博客内容的问题。

问题产生的原因非常显而易见

  • Hexo有非常方便的用SSH一键部署的方法——然而我的博客空间是传统的虚拟主机，不支持SSH
  • 未来可能转为VPS，但不是现在，现在只能用FTP
  • FTP上传速度慢且稳定性不可靠，尤其每次都上传大量的文件不可行
  • 手动管理修改内容的上传和替换太累，不符合懒人的价值观，不可行

总之为了方便的管理内容同步

  • 希望使用git来管理内容
  • 可以和Hexo内置的插件一样一键轻松部署
  • 但是不想踩ftpsync的坑

至于hexo用于FTP部署的ftpsync插件的坑是怎么回事，我没有亲身体验过。

因为我看到这个，这位博主亲身实践同步FTP会把远程端上本地没有的文件删掉。然后我想到我的FTP目录下一些本地端没有备份的零碎的东西和用来放Wordpress Blog的子目录……

怎么可能会有明知是坑还去踩踩看的勇气。

就算把数据都备份好，反复地上传和恢复数据还是会很心累。

所以另寻他法，最后就找到了这个工具：Git-ftp

对于开发这个工具的作者真是感激不尽。

Hello Hexo!

发表于 2016-07-05   |   分类于 The Journey   |  

上个星期找资料的时候，被几个界面简洁的Blog吸引，由此认识了Hexo这个静态blog页面生成框架。
然后不到一个星期我就决定抛弃wordpress转投Hexo了。
原因并不是因为用Hexo比用Wordpress看起来更高大上，也不是什么数据库臃肿访问速度缓慢之类的理由（这种没人来也经常不写的blog哪里有这种烦恼）。仅仅是因为我用markdown写完一篇笔记想po到自己blog的时候，发现号称支持markdown的WP对其的支持之差劲，甚至还会自动转移代码块里的字符，以至于无法顺利地完成编辑，折腾了一晚上无果。
于是怒从心中起。

安装过程中除了手贱没有遇到太大问题。
原本没打算太深入想分好几天来研究Hexo的安装的，毕竟不像以前爱折腾一下有的没的了，折腾其实是挺没效率的事情。一切以get things done为准。
结果没想到一气呵成。
只有部署没有使用hexo的插件，因为用的虚拟主机，SSH的设置情况不明（一時手贱甚至导致现在访问不了服务器了），ftpsync听说是个大坑，现在使用愚笨又传统的FTP上传法。
很慢，不过更多原因是因为我手贱导致无法访问服务器只好用翻窗的方法访问FTP服务器。
只不过将一大堆文件（夹）批量上传毕竟不是个可靠的做法。尤其FTP连接还十分不稳定。
打算过两天研究一下如何在本地和虚拟主机之间同步文件。实在不行的话，研究一个打包脚本也不是不可以……

还有很多插件和功能没布置好。发牢骚的日记就罢了，仅仅的几篇CS的笔记还是想重新整理一下，把从WP迁移数据的时候渲染失败的md语法修整一下。
不过回过头看看自己以前那些无聊的日志，并不觉得有什么价值。
以前的自己似乎会比现在更珍惜过去。然而却从未像现在这样觉得过去空洞且一文不值。
以前画画的圈子里，对于是否要po自己的练习图出来，看法比较两极。
一种认为，在能画到一定水平之前频繁地po图并不利于头悬梁锥刺股苦心练习，不小心自我膨胀或者不小心被打击到的可能性都大于得到交流和意见的概率。所以，少吹水po图，多画。
另一种认为，记录自己的成长和进步轨迹天经地义，利于帮助自己总结进步，分享想法，给自己一些正反馈，并没有什么问题。
然而在技术的世界里，分享的精神似乎更倾向于后者。然而我却深受前者的影响。
记忆短暂，人情缘薄。7年人生也仿佛流水般缺少记忆的实感。
虽然根本没什么好记录分享的也是个根本问题。放眼看过去都是黑历史简直羞耻心爆棚。

接下来几天打算慢慢修整和研究基本的功能。
当然还有很多其它事情要做。

一个新欢浏览器

发表于 2016-06-22   |   分类于 The Journey   |  

早上起床后打开电脑，面对昨晚浏览器里留下的一地标签页。
忽然感到心累。
每隔一段时间就会怒Opera不争「我想要这个功能想要那个功能为什么Opera里这个功能没有了这不Opera为什么我的Opera不能好好工作」的躁郁症发作了的感觉。
其实它工作得很好，很稳定。但还是会让人觉得，事情不该是这样的。因为从脑细胞乃至到操作鼠标和键盘的手指尖都会记得曾经那么一段激情而辉煌的时候。
一回想起来，就会不由得咀嚼「婚姻是爱情的坟墓」这句老话。
这个比喻好像不对。
Anyway.

我决心至少解决对标签组的需求，尝试搜索了一下Opera的扩展，并没有什么收获。于是干脆想寻找带这个功能的浏览器。
然后就找到了Vivaldi。
标签组，标签循环，丰富的自定义快捷键和鼠标手势，可定制的搜索框，键盘操作聚焦，侧栏面板，自定义标签栏位置等等。
从进主页看到第一眼的感觉，到这些所有功能，甚至社区的页面，都让人想到Opera。
没错其实Vivaldi就是一个presto Opera的精神续作。
但是这又有什么关系呢。
自从有了Chrome后，浏览器的趋势似乎就开始趋向于简洁（单？）的本体+一堆扩展来加功能了。甚至连Opera也放下身段加入了这一行列。除了能浏览网页（和并不知道有多大用处的新闻订阅）外，许多事情都要依靠插件扩展。这很Chrome，但是这很不Opera。
我们需要更多的本体的定制，自带更多的方便的功能，需要更快捷的操作，需要更有创造性的开发团队。不能凡事都依赖扩展。
Vivaldi很好，这很Opera。
玩了半小时后我已经把webkit Opera踢出了Dock和默认浏览器了。

尽管还有一些不完善的地方，和正在开发中未能利用的功能（比如邮件，虽然我倒是有点疑惑邮件面板对一个浏览器而言是不是真的必要虽说以前Opera可是连IRC都能上宁愿单独出一个桌面的邮件客户端），但是作为一个浏览器本体已经很好用了。
由于内核是Chromium，还可以依靠Chrome扩展来扩展功能。
很多事情随心所欲。

至少标签组回来了。
感觉就像找到了新的女朋友一样。
开心。

雪

发表于 2016-01-20   |   分类于 The Journey   |  

年明けから初めての雪だった。最初は土砂降りで、途中でそのまま吹雪になってしまった。気が付いたら外は真っ白になり、ダイヤモンドダストみたいに夜空からひらひらと舞い降り、とても綺麗だった。

南に生まれ育ちの私は、あまり雪を見たことはない。それでも冬はやはり雪が降らねば、と初めて実感してからもう7年も経った。昔は冬休みで静まり返った真っ白だったキャンパスを眺めながら、ただただ嬉しくてならなかった。パーティー帰りの学生のいたずらのせいで深夜1時に（中途半端に寝ているところでないことはもううちの棟のヤツらの慈悲ともいえるのだ。向こう側の棟ならよく深夜3時にセキュリティスタッフが突入する。ありがたく思え。）寮から追い出されて小雪の中に震えることは別として（あれは到底火事警報のせいだったから結局火事警報アラームの音がトラウマになった）、しんしんと降り出す雪と真っ白に覆われる風景が好ましい。

天と地の間に何もかも浄められ、余計な色彩も騒音も消され、清々しい世界になりそうだから。雪積もりは自然光を全て反射し、目線の至るところの色も明暗もいつものより柔らかく明るく見えるし。

雪が降るところを見つめると、悲しく思うのも今年は初めてかもしれない。

こんなに降り出しているのにもかかわらず、何となく音くらいは立つと思ったら、まったく聞こえずにただ単調で静寂で。

もし神様は無言劇をでも演じてくれたら、これはきっとそのワンシーンだと思う。

雪が積もったら、融けていないうちに朝一で三四郎池を観に行くって、半年前に決めたことだけれども、最近はよく眠れなかったせいで寝過ごした。もう既に日が暮れ始める頃にやっと本郷についた。加えて携帯は電池切れになってしまい、結局こんなものしか撮れていなかった。   


  

出かける時、路地に積もった雪はもう殆どかかれたか融けたか汚いものになった。それはそうでしょう。出入りに邪魔だから。それでも、「東京って忙しい町だなー」と思わずにいられなかった。どうやら変な期待を抱えてしまったようだが、それは何の期待だったか自分もよく解らなかった。ただ起きたらカーテンを開けると、これは違う、私が知っていた雪の風景ではない、と苛立った。

雪に関する思い出も沢山ある。（丸一日のテスト勉強を終えて熱が出たのにも知らずに苦痛を堪えながら同窓生のおうちから出て雪の夜を歩き抜いたことも）大学時代の思い出に重なり、つい比較してしまったかもしれない。でも、英国の雪も別にそんなに綺麗なものとも限らないのに。名古屋の雪だって…それは通学のサバイバルゲームにしか呼びようがなかった。しかも吹雪と強風付きのバーゲンセール。名古屋って、本当は風の街だね…

一体何を何と比べてしまったのでしょう。

にしても、勝手に期待を抱え込んで、また勝手に失望して、実に私らしいと思う。最近は結構気が狂っている。

こういう景色を冬の思い出として納めるのも惜しいこと。

来年また…みえるかしら。
