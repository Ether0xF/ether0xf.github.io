昨天面基

发表于 2011-08-28   |   分类于 The Journey   |  

Suki来广州实习出差，说只是想见见我。于是当然义不容辞地就去了。
由于我生物钟颠倒的缘故，出发的时候已经开始困了。娘亲送我送到花地大道，然后从西朗始发站几乎一号线坐到底，迷迷糊糊地站了大半条一号线才下车。
在天河城坐了半个下午，聊天。然后和她的实习BOSS还有BOSS的朋友一起吃晚饭。（牛肉丸火锅什么的我最近牛肉丸吃得真不少）吃完饭一路开车回芳村，然后又去在佛山的BOSS另一个朋友家参观新装修的房子。最后那个朋友送我回来，到家11点多了。
合计下快24小时没合眼，到家就扑床了。

于是其实一下午一晚上没什么特别的事情。仅仅只是聊天，不停地在聊天。
有时候大概就是缺少这么个聊天的人罢了。
不仅仅是随时可以接个话题聊，用早安开头用晚安结束。 而是因为和太多人的世界隔得太远，甚少有人可以将心比心地聊聊自己学习和生活，对以后的人参的抉择的迷茫和感情问题的看法。
甚少有人这么容易理解你和你的心情。
自己所处的圈子太小，小到令人觉得落寞。
两年不见，足以让人感慨万千，更让人觉得弥足珍贵。

Suki说，下次再见又不知道是什么时候在哪里了。
是的。一两年后，说不定又再次各奔东西。海这边，海那边，甚为遥远。

发表于 2011-08-18   |   分类于 画板   |  

其实我有时候会把他叫成三王爷。
吾家PS3君。一脸官二代。

夜渊之龙煌影就算在九曦的传说中，也是年代久远以致于无法考据的存在。煌为星辉，影为夜穹。是曾经与弥赛亚格撒尔齐驱并进的太古龙之一。在龙之战争的末期，与孪生兄弟极辉之龙共同引导麾下的子民渡海向东，调息阴阳，镇守地脉。使子民于极东之地立足立国，是为九曦。后得煌影龙血者，其名端右。
端右靖京能自永夜的深渊中唤醒煌影也许并非偶然，然而能让煌影现身现世却是龙之子心中愿望的力量使然。此后煌影便宿于龙子心中，与龙子共生死。

————————我告诉你们以上是官方版的下面来非官方版的 ————————

仗着自己是古代龙就一脸了不起的官二代。对包括现代（快灭绝了）龙在内的其他生物不感兴趣。虽然实际上的确很强大但是由于世道不济龙脉紊乱睡太久跟不上时代所以老人家有点固步自封反而不如以前那样一百零八般武艺天天换着玩儿。时常抱怨宿主吵醒自己睡觉，抱怨宿主长得矮，抱怨宿主全身上下除了那双像他的眼睛外一无是处。但是即使如此也还是偶尔会占用右文字的身体为所欲为。有时会变成小蛇藏在右文字袖子里但是有极其恶劣的咬人前科但是由于好孩子右文字斗不过他所以官二代屡咬不爽。
所以自从把他召唤出来后右文字的生活变得极其苦逼。
总是把右文字当成不经世事的小孩子于是以大人自居进行说教。可怜右文字他是个内向害羞说一不二的好孩子……
明明没有兰瑟持家好男人能干但是会毫不知廉耻地在兰瑟面前炫耀自己的能力。再说兰瑟只是龙骑士不是龙不知道三龙爷到底在炫耀啥……
唯独在某些时刻真的很强很威武很靠得住的三龙爷。

Azure Elyseano

发表于 2011-07-23   |   分类于 画板   |  

7月23日。

我已经是两月一坑了。（自嚎

这次登场的是，在天蓝的巨穹下，伫立在荷池中的板娘二号诗之。
我承认不会画水面= =
（最近传个图上BLOG都好苦逼哟……

 

全名Elyseano，嗯这名字和Elyseaon很像呐。昵称是诗之，板娘二号酱。
之前一直说她们是双子的关系——好吧，她们长得又不像。其实两人是共生关系。诗音是乐园之核，诗之则是乐园之核的制动器，也是保护机制。对乐园之核在极少情况下做出的极端和激烈行为进行制止，同时保护乐园之核不受侵扰（尤其又来保护她自毁什么的……）。因此诗之对诗音有很强烈的保护欲。不过同时在力量上也是十分强大的存在。
性格开朗活泼，又善解人意。由于乐园之核在后乐园时代的初期启动过自毁机制，为了全力制止曾经差点毁灭。应该说是世界上最关心诗音的人。

两人的生活？
和诗音一样喜欢音乐和CD。买同样的耳机和播放机，收藏双份的CD，用一样的杯子，成对的餐具。互相保持适当距离但是又十分亲近的二人组。 是个极具魄力的行动派但是却不适合学生会长style。要说起来更像社团负责人或者女子运动社团经纪人。不过要以为她会在场外亲切地递毛巾和递水就大错特错了。实际上是个很严厉的前辈。

7月的肯定坑了。而且还没决定要画谁。
