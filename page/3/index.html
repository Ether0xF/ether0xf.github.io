花開

发表于 2014-03-23   |   分类于 The Journey   |  

總之到了這個季節了。
中旬的時候因為氣溫變化比較大，暖一陣冷一陣，同一條街道上不知道誰家院子裏的櫻花早開了。
看著覺得有點可憐，因為接下來又冷了大概一兩個星期。冷風之中，對櫻花來說大概是個不太好受的天氣。
在我家門前樓梯上就能遠遠望到 這棵樹。每次下樓的時候都能遠遠望到。每次望到都發現，啊還沒謝啊。
就這樣撐過了一兩個星期，到最近天氣真正轉暖才慢慢把花開齊，然後落花，完成了一輪生命週期…… 想起來真是了不起。

附近都是住宅區，沒有高樓。
最近難得早上都醒著，沒在睡覺。
窗戶面著正東方，到了早晨六點多，日出的晨光會準時照進來。
Another good morning.

上午的時候晴空萬里。
不過因為太耀眼了，都睜不開眼睛檢查取景什麼的。
或者說，視野太開闊了感覺哪一段都挺漂亮的。

前天去大須商店街買了點東西。
也是天氣頂好的一天。

由於我之前住過的地方……南粵也好英格蘭也好，都是四季頂不正常的地方。
南粵的春天來得太早，立春開始暖，日子到了春分和清明的時候已經覺得快能做消暑的準備了。英格蘭的春天來得太晚，三四月的時候春天只是來報個到，真正要過上就要等到快夏天的時候了。
對春天的認識最多不過是天氣轉暖了學校裏開了點小花廣場上都是曬日光浴的人（和綠頭鴨）。
所以這次某種程度上算是刷新了對春天的認識。
在這個時節有該有的風物和天氣，似乎才是春天。

座落在商店街裏的富士淺間神社。
挺小的，只有這一棵櫻花樹。
不過作為駿河國的本宮淺間大社的分靈，整個名古屋也就只有三間富士淺間神社而已。
花開得很早。本地的櫻前線還沒到，看起來就至少五分開以上了。
該說不愧是供奉木花咲耶姫命的地方嗎wwwww
櫻花之神的降臨。
哎呀真想去富士本宮淺間大社看櫻花啊……可惜4月一開學，等能去的時候已經是落花的時候了。

商店街主入口前面的大須觀音。


右邊是大須觀音的本殿，右邊是十二支守本社。
香客不管哪一天都不少。

十二支守社旁邊有一個人形供奉處和齒塚。
在那裏也有一棵櫻花樹。不過才剛剛開花。

如果開得再茂盛一點，這個角度就好看了（笑

昨天去阿藍家路上。
也是不知道誰家開的玉蘭。

樹不高，但花有拳頭大小。完全遮住了門前。和周圍的樹籬還有電線杆比較一下……
轉進這條街道時立刻就能看到。
實在是有點驚艷。

下星期抽個空去看看熱田神宮。
然後等滿開的時候就可以去鶴舞公園拍櫻花了。
春天真忙。

漢方藥局

发表于 2014-01-27   |   分类于 百字册   |  

某天經過上下學路過的漢方藥局時，忽然腦洞大開，想出了這麼一個片段。
估計是因為這邊的中華相關的店不管是藥局還是飯店，都有種莫名的內向的神祕感吧。
僅此記錄。

－－－－－－－－－－－－－－－－－－－－－－－－－－－－

　　每天上下學，我都要經過家對面的一個頗大的街區。從外面看的話那地方盡是住宅，但是稍微往裡走的話，隨著店鋪的增多，就開始有了在逐漸接近商業區的感覺。
　　在狹窄的街道兩旁，公寓的間隔中林立著大大小小的店鋪。有附帶大停車場的書店和和果子專賣商店，也有十分有生活氣息的花屋、咖啡館和壽司店。在街道的轉角一拐過去，還有極小的豬肉蓋飯便當屋這樣的偶見的驚喜。
　　總之，是十分有生氣的地方。
　　但是在這些鮮明的店鋪中間，某一天我忽然發現了一間漢方藥局。
　　黑色的方柱做成橫欄裝飾著門面。灰色的牌匾上寫著漢方藥局四個大字。
　　那間藥局，有開店營業過嗎？
　　和電車站站前那家漢方藥鋪不一樣，沒有暖色原木和中華式大氣奢華的裝潢，也沒有偶爾從撐起的木窗後面飄出的中藥味的蒸汽。那這家藥局那黑色木製的門面，彷彿就是為了隱藏它而設的一般。
迄今為止我應該騎著自行車從這家店面前路過了無數次吧。因為，那間附帶停車場的大書店可就在旁邊喲。我一定路過了無數次，沒錯。
　　但是迄今為止，從未注意過它。要說為何的話，大概是這家店看起來似乎從沒有開店營業過吧——就是說，沒有人活動的氣息。
　　在熱鬧來往的街道上，絲毫沒有人的氣息的，大概就只有這一家了。
　　今天路過的時候……為什麼我竟然注意到了呢？
　　對了……因為今天，藥局裡點起了燈。

　　今天是陰雨天，厚厚的雨雲遮蔽了整個天空，天色就彷彿夜晚般陰暗，讓人不由得懷疑天上那大團的棉花裡吸滿了墨水而不是雨水。大雨剛過，來往車輛和行人毫不留情地碾過路面的積水的「啪沙——」的聲音不絕於耳。所有人都想趁著雨勢式微的時候趕緊趕往下一個目的地，天和地之間的世界，似乎都因為這場雨而變得慌惶起來。
　　連綿的陣雨淅瀝淅瀝地填充著焦慮的環境雜音的縫隙，我在剛剛轉變成紅燈的人行橫道前急剎，將自行車聽了下來。
　　討厭，又要在雨中多淋一會兒了。今天就不應該騎自行車上學的。
　　我在路口張望起來，呼出的氣息變成了眼鏡片上的白霧。
　　就是這個時候，我看到了道路對面的漢方藥局。
　　在屋檐上滴滴答答的水簾後面，昏黃的暖光從店面的玻璃後面透出來，倒映在路面的積水上。
　　啊啦，那家店……在營業？

【紐約時報】撫慰傷者是門藝術

发表于 2014-01-26   |   分类于 The Journey   |  

不要说“事情会过去的”。凯瑟琳写道，“‘事情会过去’的说法是不成立的，大的伤痛会在发生之后形成新的常态。不会‘变回过去的我’。”

不要说什么这也许不那么糟，或者试图理解发生的一切。凯瑟琳及父母说话的时候，语气异常温和，也经过冷静思考，不过很明显的是，这些悲剧使他们不能容忍装腔作势和盲目乐观。

從紐約時報中文網上看到的。
这篇是专栏作者DAVID BROOKS撷取Catherine Woodiwiss为《寄居者》(Sojourners)杂志撰写的一篇博客文章“A New Normal: Ten Things I’ve Learned About Trauma”总结撰写而成的。
当然绝不是心灵鸡汤的十大系列那种废话。
因为原文是英文，有条件看懂的可以看看。
之所以转载这篇文章是因为，只是剪贴保存不足以表达我对这篇文章的赞同。它为伤痛者的感受说话。
通篇看完后，大概会感受到，那些受到过伤痛的人再一次一次被推向深渊，都是因为周围人的披着爱和谊以及各种名义幌子的愚蠢。
他们觉得孤独，并要孤独地一个人承受和经历这些，甚至比一个人承受更糟因为他们的独处空间没有受到足够的尊重，有时候都可能来源于周围自以为是且无知、冷漠和完全错误的爱。
愚蠢至极。
每个人或多或少都会犯过的愚蠢。

新年

发表于 2014-01-05   |   分类于 The Journey   |  

明天開學。

新年沒什麼感覺就到來了。
在阿藍家吃了一晚上火鍋看紅白，然後看著電視玩了通宵。
早上天剛亮的時候，才騎車回家睡覺。
微微地下著雨，薄薄的天光，寒冷的空氣裡有著冬天特有的衰草寒煙的味道， 安靜的街道。
到處都瀰漫著一種寂靜卻嶄新的氣息。
普通，但卻最好不過的早晨。
今年之計，也會不錯吧。讓人忍不住如此覺得。

最近想買張床了。
一開始覺得直接鋪上墊子睡在地板上就好，現在其實也覺得這樣睡並沒有任何不便。
但是開始感覺到1DK的洋室裡沒有床是多麼不便的事情。
說實話現在房間裡很亂。雖然有空間，但是空間卻被大把地浪費著，也沒法好好地收納和整理物品。 首先如果有床的話，應該馬上可以給房間劃分出一個格局來吧，因為房間裡床應該是佔地最大的傢俱了，也是整個房間的生活中心。然後剩下的空間裡，哪裡擺什麼，就能分得很清楚了。床底下的空間還可以用來放兩個大行李箱。這樣將牆壁內櫥整理一下的話，還可以空出很多空間來。還有幾個架子也是需要的，雖然東西不多，但是還是需要能擺放的地方。一層放電飯煲和電熱水壺，一層放收納雜物的盒子，一層放書之類的物品，大概三四層就夠了。
可以預見房間會變得整齊有序，也便於打掃的樣子。

雖然這些不是一定必要的，但是卻是需要的。
要稍微畫點心思去打理，大概這樣就是生活吧。

說到底……還是想要一個安穩的居所啊。
居所如果不安穩的話，似乎連讓人放開手腳去打理生活的心情和勇氣都沒有。總是要考慮到搬遷時的麻煩。大概也有可能是大學住宿舍不斷搬動的緣故留下來的毛病。
當然，也是因為我潛意識裡的緊張和不安。雖然我算是個隨遇而安的人，對環境的適應力還算強。不過始終卻不是很喜歡到處挪動。

就這樣……這幾天耐心地看看傢俱好了。
