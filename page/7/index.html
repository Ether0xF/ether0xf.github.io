汽水

发表于 2012-12-16   |   分类于 The Journey   |  

今天下午去Tesco買東西，路過放汽水的架子。

之前都沒有在本地見過賣沙示汽水，今年突然發現了屈臣氏的沙示汽水。

這樣的……

因為好奇就買了一罐來喝。

剛剛洗完澡後打開喝了一口……後，表情➝

誠如網上部分人所言，這味道簡直是風油精……氣比可樂還要足，喝完後嘴裡那種汽水的甜味比可樂還要重，弄得我現在狂喝白開水洗掉嘴裡的味道。

直到喝完都還是不喜歡。還是比較喜歡Dr. Pepper可惜這邊沒有賣。

去搜索了一下沙示汽水，原來還非常有名。在廣州也是盛行已久的樣子（廣州人的颶風君表示喜歡）。

以下是摘自維基百科 - 沙示的條目。

沙示（Sarsaparilla或Sarsae）是一种碳酸饮料，以植物Sarsaparilla（墨西哥菝葜）为主要调味的原料，因此得名。为深褐色、甜味、不含咖啡因。色泽相近于可乐但口味截然不同。

　　该饮料来源不可考，应该是墨西哥原住民用来解热的自然草本饮品。美国人移民新大陆，且往中美洲扩展势力范围后，成为美国民众常喝饮料。据史载，19世纪，沙示与姜汁水为美国人常喝的两种解渴饮料。可乐发明人约翰·彭伯顿（John Pemberton）当初调配具有古柯碱成分的可乐，其用意之一是为了取代当时盛行于美国的沙示。

　　有资料显示，早于二十世纪初屈臣氏就已经开始在上海等地生产当时名为“荷兰水”的沙示汽水[3]。1930年代，曾于广州屈臣氏汽水厂工作的李智扬与李冠玲、梁汉奇、黄有桐、王贞三等11人集资筹办汽水厂。他们购置了一台旧机器，于1946年2月在大德街80号的一家300平方米厂房正式开业，名为亚洲汽水。

　　1949年后，屈臣氏退出大陆市场。亚洲汽水的厂址迁到大德路298号2000多平方米的厂房。当时亚洲汽水拥有3台12头汽水机，职工80多人，年产量达500多万瓶。1956年3月，亚洲汽水厂实行公私合营。在1960年代中期，曾生产各种口味的汽水并出口香港、澳门等地。文化大革命时亚洲汽水一度易名，文革结束后才改回原来名字。至1990年代初，亚洲汽水在全国有多家厂房。但1991年公司股权出售给百事可乐公司，随即逐渐式微。2009年5月1日，亚洲汽水被香雪制药接管。

至於墨西哥菝葜又是什麼東西……也Wiki了一下。

菝葜属植物为攀缘植物，在世界各地均有分布，约有350种。自古以来便被视作香草、野菜和生药、中药，并用于治疗风湿、痛风、感冒以及用于退烧。欧洲从16世纪起也把菝葜属植物作为淋病和梅毒等性病的治疗药方使用。因其含有较多的皂苷等配糖体，也有人视作类固醇药剂来使用以暂时提高运动能力。以及作为治疗其他皮肤疾病（如：脓疮、皮肤糜烂）、水银中毒等药的材料之一来使用。主要使用其干燥处理后的根部和根茎部分。

在墨西哥菝葜製作的飲料的列表裏，赫然發現了Dr. Pepper……略為驚訝。
喝了兩年還是第一次知道。
不過Dr. Pepper是初喝不喜歡之後就會覺得很好喝的汽水。
但是全然不覺得沙示會合我的口味= =

順便在超市看到這個。

可口可樂和百事好基友一輩子。
而且極度不管瓶裝還是罐裝的包裝手感好像很高級。
不過大家都覺得很難看很土……用颶風君的話說就是工業設計都死絕了。

騎自行車

发表于 2012-12-15   |   分类于 The Journey   |  

自從區裏設置了公共自行車後一直想試試看，但是騎自行車已經是十幾年前的事情了。雖然有了IC卡但是因為不會騎加上本地的道路上很少有自行車專用道騎車有點危險所以一直沒用過……
前天走路回家的時候路過公園旁邊的自行車的取放點時，也不知道怎麼心血來潮就取了一輛來騎了。
……騎上去後覺得，真是和冒險差不多。

沿著一中門口的路騎過去，雖然有自行車道但是只有窄窄的一條，而且還有路邊還有車子停放在自行車道上|||幸好這條路上汽車不多。不過還是好幾次剎車停下來重新調整速度和姿勢。
車頭感覺很重，方向和速度都很難控制。
遇到十字路口的話還是會乖乖下車推著走……
從一中門口然後走後面的綠田路，途中經過一所幼兒園正好是放學時刻車子很多，也只好下來推著走。
看到騎著自行車和摩托車的人熟練地從身邊開過在車流裏穿行真是佩服。

然後從綠田路出來，把車子停放到醫院門口的取放點。大概只有10分鐘的腳程。
雖然只是很短的一段路，但是有種冒險的感覺……
騎在車上有種身體還記得怎麼操作的感覺。

下次路上人少的話……再騎一次吧。

不思議遊戲個人角色簡（亂）評——青龍篇·本鄉唯

发表于 2012-06-18   |   分类于 百字册   |  

唯绝对不是一个三言两语能概括的角色，也不是几千字能评析完的角色。去了解这个角色，除了观察她行动的每个细节和表情，甚至想象她在你面前的样子，大概别无他法。唔，我无法否认这句话带有相当的个人YY的成分。但是确实如此，因此直接的描写比较欠缺，隐藏在故事中的细节和信息都需要挖掘并重组，才能解读出这个故事另一面的样子（我又恰好乐意干这种事情）。而且前面写朱雀组的几位角色的简评时，我都尽量从剧情和故事本身秉持尽可能的客观去挑一个我喜欢的角度评析。但是只有写到唯的时候，我发现大脑是没有「客观」可言的。最近开始写青龙的同人的缘故，急切需要整理出思路以把握角色特质，于是便从青龙的巫女开始构思青龙组的角色简评。但是我发现这太难，有很多很多论点在我脑海里打转，想到了一个写了点又发现漏了另一个，一直都布不出一个基本的文章结构。文章没有基本的结构和条理是难以下笔的，以至于这篇简评被我一拖再拖，以至于这后面有很多很多的坑还在坑着。

没办法……她是我在这部作品里唯一，甚至是我看过的其他所有作品里唯二，说得上「怜爱」两个字的角色。我可能会为一个精彩的悲剧里的某个角色感到惋惜或者感动和敬意，但是只要他们都为剧情和故事服务了，我都不会感到丝毫的一点点可惜，毕竟故事才是最重要的。不过怜爱一个角色，就是会去研究这个角色方方面面，会不自觉地站在角色的角度去看待问题，会哪怕变成悲剧的话故事会比较好更有看头也不希望拿这个角色去牺牲，而是更希望这个角色获得幸福。

所以类比一下，这才能了解其他人那么喜欢星宿柳宿等等角色的心情。

所以，接下来的乱语会相当多，甚至可能还有掉节操的部分（揍）。于是结束乱语的闲谈，进入乱语的正题吧。
